# Web版PDF页码添加工具

一个纯前端的PDF页码添加工具，无需上传文件到服务器，完全在浏览器中处理，保护隐私。

## 功能特性

- **拖拽上传**：支持拖拽或点击选择PDF文件
- **页码位置**：6种预设位置 + 自定义百分比位置
- **多种格式**：数字、罗马数字、字母等多种页码格式
- **自定义样式**：可调整字体大小和颜色
- **实时预览**：支持PDF预览和页码效果预览
- **隐私安全**：所有处理在本地浏览器完成，不上传文件

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 上传PDF文件
3. 根据需要设置页码位置、格式、字体大小和颜色
4. 点击"生成预览"查看效果
5. 点击"添加页码到PDF"处理文件
6. 点击"下载"保存处理后的PDF

## 技术栈

- PDF.js - PDF渲染
- pdf-lib - PDF页码添加
- 原生JavaScript + CSS
